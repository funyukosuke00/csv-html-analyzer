# CSVデータ分析ツール（Python）

CSVデータを読み込み、男女別の平均点を計算し、
結果をHTMLとして自動出力するPythonツールです。

## 使用技術
- Python
- pandas

## 機能
- CSVファイルの読み込み
- 男女別平均点の計算
- 平均点の比較
- 結果をHTMLとして出力

## 実行方法
1. gender_avgscore.py を実行
2. gender_avgscore.html が自動生成されます

## ファイル構成
- gender_avgscore.py：メインプログラム
- practice20260401.csv：サンプルデータ
- gender_avgscore.html：出力結果
